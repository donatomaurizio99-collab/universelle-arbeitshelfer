# Universelle Arbeitshelfer

Statische GitHub-Pages-Version der Dokumentation "Universelle Arbeitshelfer v1.1".

## Aktualisieren

1. Neue Komplett-HTML erzeugen.
2. Die neue Datei als `index.html` in dieses Repository kopieren.
3. Committen und nach GitHub pushen.
4. GitHub Pages veroeffentlicht die neue Version automatisch.

Die HTML-Datei enthaelt keine Skripte und fuehrt selbst nichts aus.
